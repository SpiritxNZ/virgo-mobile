# Readme

## 目录

### assets

放静态资源

### components

存放无状态视图组件(styledButton,styledText之类的)

### components/layout

存放layout组件

### contants

存放常量

### navigation

存放导航组件

### screen

存放有状态的组件

### action

存放redux的action

### reducer

存放redux的reducer

### service

存放通用方法

### service/api

存放api请求

## 命令

在package.json下的scripts里

## 插件

这是一个用expo管理的react native项目，尽量使用expo自带的react native插件。状态管理尽量用redux。新加插件时在下面列出

用到的其他react插件：
  redux, react-redux, moment, axios, redux-logger, redux-thunk

用到的其他react-native插件：
  react-native-paper
